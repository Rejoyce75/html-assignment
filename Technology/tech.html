Differences between paddings, margins, and borders

padding

An element's padding is the space between its content and its border. It creates extra space within an element.

Margin

This is the space around an element. Margin creates extra space around an element. The margin property may be specified using one, two, three, or four values. Each value is a length, a percentage, or the keyword auto. Negative values draw the element closer to its neighbors than it would be by default.

Border

The border-style property specifies what kind of border to display. This property is a combination of three other properties border-width, border-style, and border-color as can be used as a shorthand notation for these three properties. It sets or returns the border-width, border-style, border-color Properties.
